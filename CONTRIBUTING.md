# Contributing

Thanks for the interest in Grain.

This repository is currently public for visibility, but external contributions are **not** being accepted.

## Pull Requests
Please do not open pull requests for features, refactors, or fixes unless you have been explicitly asked to do so by the repository owner.

Unsolicited pull requests may be closed without review.

## Forks
GitHub may allow public forks for technical reasons, but that does not grant permission to reuse or redistribute the code. See [LICENSE.md](c:/Users/tlton/OneDrive/Desktop/grain/LICENSE.md).

## Issues
If issue tracking is enabled, issues should be limited to:
- reproducible bugs
- critical security concerns
- clear product-breaking regressions

## Licensing / Collaboration
If you want to discuss collaboration, licensing, or commercial use, contact the repository owner directly.
